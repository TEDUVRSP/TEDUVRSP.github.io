About Our Project

"VR Project Blue" was born to be an entertaining and educational option for individuals who want to enter the cyber security field of software engineering using VR technology. The basics of cyber security will be taught to users with a serious game that can be played with a virtual reality glasses headset on a computer with the necessary system features.

  • Brand new serious game for cyber security.
  • One of the rare games that teach cyber security and the only game that uses VR.

The player will find himself in a study room belonging to the character he directs. He will choose a task for himself from a social platform that includes well-meaning freelance cybersecurity experts, called white hat hackers. This task can be one of various tasks such as finding security vulnerabilities of companies, penetrating testing, testing the limits with DDoS attack. The player will understand the requirements of the task and reveal what needs to be done. With the character he manages, he will first code the software necessary for his task. This will complete the coding by selecting the correct code blocks or dragging and dropping them where they need to be. If the task requires any physical contact, the player will be able to instructively experience different social engineering methods such as usb stick fishing rod or infiltrating the server room with VR technology. With the different scenarios presented in different missions, the player will learn how and with what methods cyber attacks can be made and what security methods should be used to prevent them by experiencing a fun serious game.
